## 简述

Fluent Bit 是一个快速、轻量级的日志处理器和转发器，专门为嵌入式设备和边缘计算设计。本文档涵盖了 Fluent Bit 的基础概念、常见问题处理以及实际生产环境中遇到的故障排查案例。
